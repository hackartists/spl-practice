* Deployment
+ It shows =Program Id= and update =PROGRAM_ID= file.

#+begin_src bash
  make deploy
#+end_src

* Upgrade
+ If you want to save logs, you can set =VERSION= environment.
  + If you set =VERSION=, it write a log file in =build/program-{VERSION}=.

#+begin_src bash
   make upgrade
  # VERSION=2 make upgrade
#+end_src
